# Network Addresses

Get access to all the available networks.

## Installing

using npm
```bash
npm install @emmsdan/network-address
```

using yarn
```bash
yarn add @emmsdan/network-address
```

## Usage

```javascript
// CommonJS
const networkAdd = require('@emmsdan/network-address').v4;


// ECMA Script
import { v4, v6 } from '@emmsdan/network-address';

```
